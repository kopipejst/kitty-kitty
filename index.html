<kitty-kitty>

A web component for adding random cat pictures to webpages

Refresh the page for another kitty!

How to use kitty-kitty

Include HTML Import tag

<link rel="import" href="webcomponent/kitty-kitty.html">

Then place cats where ever you want!

<kitty-kitty></kitty-kitty>

We have parameters

Decide what types of images you'd like to show.
Accepted types are JPG, PNG and GIF.

<kitty-kitty type="gif"></kitty-kitty>

Limit image dimension.
Small (250px), Med (500px) & Full (original size).

<kitty-kitty size="med"></kitty-kitty>
